In -rw-r--r-- 1 userhacsa userhacsa 70 Oct 14 10:00 script.sh
-rw-r--r-- is the file permission
rw- means the owns can only view the contents and make changes to the file 
r-- shows that the group can only view the content of the file
r-- shows that other users can only view the content of the file
- shows the file is a regular file 
1 userhacsa is the name owner
userhacsa is the name of the group
Oct 14 10:00 is the date and time which the file was last modified
1 is the number of hard links to the file
70 is the size of the file in bytes
script.sh is the file name
